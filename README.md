# 📊 E-Commerce Public Dataset Dashboard

## Setup Environment - Shell/Terminal
```
pip install pandas numpy matplotlib seaborn streamlit babel
```

## Run steamlit app
```
streamlit run dashboard.py
```

## Tujuan Analisis
### Pertanyaan 1: Tren Performa Penjualan Bulanan dari Tahun 2016-2018
Tujuan analisis ini adalah mengidentifikasi pola musiman dan tren pertumbuhan bisnis dari tahun 2016-2018. Teknik yang digunakan adalah Time Series Analysis dan Seasonal Analysis.

### Pertanyaan 2: Segmentasi Pelanggan dengan RFM
Tujuan analisis ini adalah mengelompokkan pelanggan berdasarkan perilaku pembelian berdasarkan metode RFM (Recency, Frequency, Monetary). Teknik ini menghasilkan 5 segmen yaitu Best Customer, Loyal Customer, Regular Customer, At Risk Customer, dan Lost Customer.

## Fitur Dashboard
- Key Performance Indicators (Total Revenue, Total Orders, Average Order Value, Unique Customers)
- Grafik jumlah order dan revenue per bulan
- Geographic Sales Analysis (Top 8 state dengan revenue tertinggi)
- Customer Segmentation (distribusi 5 segmen pelanggan)
- Distribusi RFM (Recency, Frequency, Monetary)
